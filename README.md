Opción 1: VS Code (si lo tenés)
Instalá la extensión Live Server (botón Extensions, buscar "Live Server")
Clic derecho en index.html → Open with Live Server
Se abre en el navegador en http://localhost:5500

Opción 2: Python (viene instalado en Mac/Linux)
Abrí la terminal
Navegá a la carpeta del proyecto:
cd ruta/a/tu/carpeta
python3 -m http.server 8080
entrar a: http://localhost:8080
